## TC-LOGIN-02 - Login con password inválido

**Precondición:**
Usuario registrado

**Pasos:**
1. Ir a la página de login
2. Ingresar usuario válido
3. Ingresar contraseña inválida
4. Click en login

**Resultado esperado:**
El usuario no debe acceder al sistema
Se debe mostrar un mensaje de error indicando que la contraseña es incorrecta
El usuario permanece en la página de login
